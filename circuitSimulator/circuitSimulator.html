How this simulator works:
This simulator works by using KCL and nodal analysis. It essentially is a simple DC simulator which has been adapted to allow for AC functionality, albeit inaccurate. Inductors find the current flowing through them for the next simulation frame by finding the voltage across the component and incrementing the current accordinly. Capacitors find the voltage across them for the next simulation by finding the current flowing into them and using V=Q/C. This is by no means a proper simulator, but really just an excersize forcing me to learn more about circuits and how they work.
