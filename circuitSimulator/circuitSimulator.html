How it works:
To start, this is by no means a proper circuit simulator. It lacks functionality and some accuracy, although the entire purpose of the project was to force me to learn more about circuits and circuit analysis. This simulator works by using KCL and nodal analysis. It essentially is a simple DC simulator which has been adapted to allow for AC functionality, albeit likely a bit inaccurate. Inductors find the current flowing through them for the next simulation frame by finding the voltage across the component and incrementing the current accordinly. Capacitors find the voltage across them for the next simulation by finding the current flowing into them and using V=Q/C.

Notes:
-- Inductors have a tendancy to make voltage blow up to infinity. Watch out for inductors in series or parallel.
-- Small inductor and capacitor values are harder to simulate. The time step needs to be adjusted if using smaller values.
