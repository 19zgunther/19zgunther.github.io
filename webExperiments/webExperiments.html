This project started off as simply trying to render a basic cube but quickly blew up into a reflective graphics experiment.

In this scene there are four different light sources: two ambient and two reflective. The ambient light sources work by taking the dot product between the normal vector (for each triangle) and a static vector which represents the direction the ambient light is coming from. If the normal and the static vector are aligned or close to parallel, then the surface is illuminated and the color of the triangle is displayed on the screen. As the vectors point farther apart, the light level for the triangle is decreased until no color (only black) is added to the screen.

The reflective light sources are simply large spheres located far away from the objects in the scene. If the normal vector (for each fragment of each triangle) points towards & passes through the spheres, the color of that fragment is multiplied by a scalar (determined by the normalized distance the ray travels inside of the sphere) which has the effect of making the objects look reflective.
